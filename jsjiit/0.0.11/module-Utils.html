(static) generate_date_seq(dateopt) → {string}

Generates a date sequence string from a given date or current date The sequence format is: d[0]m[0]y[0]w[0]d[1]m[1]y[1] where: d = day padded to 2 digits m = month padded to 2 digits y = last 2 digits of year w = weekday number (0-6)
Parameters:
